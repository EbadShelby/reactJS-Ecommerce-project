# reactJS-Ecommerce-project
this repo is created to practice my Javascript and reactJS skills and so on.

## Notes:

### Vite folders:
**Node Modules** - where npm install all the packages.
**src** - codes for your website.
**eslint** - highlights problem in JS codes.
**esLint.config.js** - configure esLint.
**package.json** - list of packages we need to install (npm install).
**package-lock.json** - save versions of packages installed.
### React:
**useState** - reactive data.
**useRef** - DOM refference.
**useEffect** - reacts to changes or events outside React.
**< StrictMode >** - additionl checks and warnings when developing our app.
**Routing** - create multiple pages using 1 html file.
**BrowserRouter** - wraps everything that uses routing.
**Routes** - all pages of our website.
**Route** - add a page.
**< Link to="" >** - switch page without reloading.
