# reactJS-Ecommerce-project

this repo is created to practice my Javascript and reactJS skills and so on.

## Notes:

### Vite folders:

<p>**Node Modules** - where npm install all the packages.</p>
<p>**src** - codes for your website.</p>
<p>**eslint** - highlights problem in JS codes.</p>
<p>**esLint.config.js** - configure esLint.</p>
<p>**package.json** - list of packages we need to install (npm install).</p>
<p>**package-lock.json** - save versions of packages installed.</p>
### React:
<p>**useState** - reactive data.</p>
<p>**useRef** - DOM refference.</p>
<p>**useEffect** - reacts to changes or events outside React.</p>
<p>**< StrictMode >** - additionl checks and warnings when developing our app.</p>
<p>**Routing** - create multiple pages using 1 html file.</p>
<p>**BrowserRouter** - wraps everything that uses routing.</p>
<p>**Routes** - all pages of our website.</p>
<p>**Route** - add a page.</p>
<p>**< Link to="" >** - switch page without reloading.</p>
