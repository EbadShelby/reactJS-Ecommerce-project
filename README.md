## Notes:

### Vite folders:

<p><b>Node Modules</b> - where npm install all the packages.</p>
<p><b>src</b> - codes for your website.</p>
<p><b>eslint</b> - highlights problem in JS codes.</p>
<p><b>esLint.config.js</b> - configure esLint.</p>
<p><b>package.json</b> - list of packages we need to install (npm install).</p>
<p><b>package-lock.json</b> - save versions of packages installed.</p>
### React:
<p><b>useState</b> - reactive data.</p>
<p><b>useRef</b> - DOM refference.</p>
<p><b>useEffect</b> - reacts to changes or events outside React.</p>
<p><b>< StrictMode ></b> - additionl checks and warnings when developing our app.</p>
<p><b>Routing</b> - create multiple pages using 1 html file.</p>
<p><b>BrowserRouter</b> - wraps everything that uses routing.</p>
<p><b>Routes</b> - all pages of our website.</p>
<p><b>Route</b> - add a page.</p>
<p><b>< Link to="" ></b> - switch page without reloading.</p>
